# Data Sources

Below are the primary data sources powering BlueSphere. We aim to use **open, reputable** datasets with clear licensing.

> Licensing summaries are provided for convenience — refer to the source sites for definitive terms.

## Catalog

| Source | Dataset | Update Cadence | Access | License |
|---|---|---|---|---|
| **NDBC (NOAA)** | Realtime2 buoy observations (WTMP) | Hourly | <a href="https://www.ndbc.noaa.gov/" target="_blank" rel="noreferrer">Website</a> | Public domain (US Govt) |
| **ERSST v5 (NOAA/NCEI)** | Monthly global SST (and anomalies) | Monthly (~1 mo lag) | <a href="https://www.ncei.noaa.gov/products/extended-reconstructed-sst" target="_blank" rel="noreferrer">Product page</a> | Public domain (US Govt) |
| **ERDDAP (NOAA CoastWatch etc.)** | Tabledap/Griddap datasets (e.g., SST) | Varies by dataset | <a href="https://coastwatch.pfeg.noaa.gov/erddap" target="_blank" rel="noreferrer">Portal</a> | Source-specific, often public domain |

## How We Use These
- **Ingestion** converts files/feeds into a normalized format and **upserts** to our database.
- We precompute **tiles** for map rendering (SST PNG tiles, currents MVT).
- See <Link href="/docs/DATA_SOURCES">DATA_SOURCES.md</Link> and <Link href="/docs/DATA_LICENSES">DATA_LICENSES.md</Link> for details.
